CV.Build is a free frontend web application that helps Nigerian job seekers 
create professional CVs in minutes.

Users enter their job role, fill out a detailed form with their work experience, skills, education, and more. The app randomly selects 3 unique CV templates from 
a pool of 12 professionally designed layouts. Meaning every user sees a different combination. Users pick their favorite and download it as a PDF instantly.

Built with: HTML, CSS, Vanilla JavaScript
Fonts: Cormorant Garamond, Libre Baskerville (UI) + mixed serif/sans-serif per template
No backend. No database. No payment. Runs entirely in the browser using localStorage.

Pages:
- index.html      — Landing page with job role search
- question.html   — Detailed CV form
- templates.html  — 3 randomly selected templates from pool of 12
- download.html   — Preview and download as PDF
